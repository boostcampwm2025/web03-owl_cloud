## 🛎️ 서비스 소개

> 서비스 URL: https://www.clobby.p-e.kr/

`"화상 회의 따로, 코드 공유 따로... 번거로운 협업은 이제 그만."`

dev:meet는 화상 통화, 실시간 코드 에디터, 화이트보드를 하나의 브라우저 탭에서 완벽하게 통합한 개발자 전용 협업 솔루션입니다.

프로젝트나 회의를 진행할 때 우리는 Zoom에서 얼굴을 보고, VS Code로 코드를 공유하며, FigJam으로 다이어그램을 그리곤 합니다. 하지만 이러한 도구의 파편화는 다음과 같은 불편함을 야기합니다.

- 몰입 방해: 툴 간의 전환(Context Switching)에 따른 피로도와 논의의 흐름 단절

- 환경 설정의 번거로움: 페어 프로그래밍을 위한 복잡한 환경 설정 및 세션 공유 과정

- 시각화의 한계: 텍스트 기반 코드 블럭만으로는 설명하기 어려운 아키텍처 및 로직 설계

**dev:meet는 이 모든 과정을 하나로 합쳐, 오직 '개발'과 '소통'에만 집중할 수 있는 환경을 제공합니다.**

<br/>

### 서비스 핵심 기능

1. 그룹 화상 회의: WebRTC(SFU) 기반의 고화질/저지연 화상 통화 및 화면 공유

2. 동기화 코드 에디터: Monaco Editor 기반의 자동 완성, 에러 감지 기능이 포함된 실시간 공유 에디터

3. 공유 화이트보드: Canvas API 기반의 다이어그램 드로잉 툴로 아키텍처 설계 및 브레인스토밍 지원

4. 실시간 채팅: 회의 중 자료 공유 및 빠른 피드백을 위한 메시징 기능
