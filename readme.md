## 🛎️ 서비스 소개

- 서비스 URL: https://www.clobby.p-e.kr/

- 서비스 클로비(Clobby)는 "나만 알기 아까운 취미와 취향"을 한 장의 카드로 기록하고, 타로 카드를 뽑듯 타인의 취미와 취향을 우연히 발견하는 초경량 취향 기반 카드 플랫폼입니다.

- 단순한 기록을 넘어 카드제작 및 실시간 ‘챌린지’를 통해 같은 취향을 가진 사람들과 느슨하게 연결되며, 나만의 취향이 누군가의 새로운 시작이 되는 선순환 구조의 취향 허브를 지향합니다.

### 서비스 핵심 기능

1. 나만의 미니 캔버스, [카드 제작소]

   - 자유로운 레이아웃: 텍스트, 이미지, 영상, 스티커 등 다양한 요소를 드래그 앤 드롭으로 자유롭게 배치하여 한 장의 카드를 디자인합니다.
   - 창작의 즐거움: 정형화된 피드 양식에서 벗어나 폰트, 컬러, 테마를 활용해 자신의 취향을 시각적으로 구현하는 ‘다꾸(다이어리 꾸미기)’ 감성을 제공합니다.

2. 우연한 만남의 미학, [카드 뽑기]

   - 인터랙티브 탐색: 모든 카드가 뒷면으로 배치된 상태에서 사용자가 직접 카드를 선택하고 뒤집는 ‘타로 뽑기’ 방식의 인터랙션을 제공합니다.
   - 다양한 추천 로직: 인기 카드 위주의 랜덤뿐만 아니라, 내가 몰랐던 분야를 추천하는 ‘관심사 외 랜덤’ 등을 통해 탐색의 외연을 넓혀줍니다.

3. 실시간 취향 공유, [라이브 챌린지]

   - 함께하는 취미: 방장이 챌린지 카드를 생성하면 정해진 시간에 라이브가 열리며, 시청자는 ‘참여 요청’을 통해 실시간으로 자신의 활동을 공유할 수 있습니다.
   - 휘발성과 기록의 공존: 라이브 세션은 종료 후 삭제되어 참여 부담을 낮추되, 반응이 뜨거웠던 순간은 ‘하이라이트 클립’으로 자동 생성되어 프로필에 기록됩니다.

4. 느슨한 연결과 확장, [구독 및 리워드]
   - 취향 구독: 마음에 드는 카드 제작자를 구독하여 새로운 취향 소식을 빠르게 받아보고, 댓글과 좋아요를 통해 부담 없는 소통을 이어갑니다.
   - 활동 보상: 카드 제작, 챌린지 참여 등으로 얻은 포인트로 카드 제작소의 전용 꾸미기 아이템이나 스티커를 해금하여 자신의 개성을 더 강화할 수 있습니다.
