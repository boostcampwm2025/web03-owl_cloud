## 1. root 

- 권한: 모든 권한을 다가지고 있음

## 2. admin

- 권한: root에 버금가는 권한을 가지고 있음
- 제한: localhost 환경에서만 접속이 가능하다. (deploy에만 존재)

## 3. test_db_dev

- 권한: test_db를 실제로 사용하는 유저
- 제한: test_db에 서비스를 이용하는데 사용되어 지는 유저 ( 테이블 자체를 생성하고 만드는 건 안된다. )

## 6. deploy_db_admin

- 권한: deploy_db에 대한 모든 권한을 가지고 있는 유저
- 제한: deploy_db에 대해서만 모든 권한을 가지고 있다. 

## 7. deploy_db_dev

- 권한: deploy_db를 실제로 사용하는 유저
- 제한: deploy_db 서비스를 이용하는데 사용되어 지는 유저 ( 테이블 자체를 생서앟고 만드는 건 안된다. )

## 8. deploy_db_view

- 권한: deploy_db 쿼리 문을 보고 긁어올수 있는 권한을 가진 유저
- 제한: 읽는 것만 가능하다.
