# vpc

## vpc 이름

### owl-cloud-vpc
- 설명: owl-cloud가 사용하는 vpc

## subnet 종류

### owl-cloud-test-subnet
- 테스트 서버가 사용하는 서브넷

### owl-cloud-deploy-subnet
- 배포 서버가 사용하는 서브넷

### owl-cloud-infra-subnet
- 설명: infra 서버가 사용하는 서브넷 (private)

## server 종류

### owl-cloud-infra-server-1 (private)
- 설명: owl-cloud에서 test-server가 사용할 infra 서버 이다. ( 협소하다면 deploy도 사용할수도 있다. )

### owl-cloud-test-server
- 설명: owl-cloud에서 테스트 서버가 위치하는 서버
